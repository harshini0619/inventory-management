Inventory Management System
Inventory Management System is a web-based application designed to simplify inventory tracking and product management. It provides an efficient way to manage product details, monitor stock availability, and organize inventory operations through a simple and user-friendly interface.

Live Demo
https://harshini0619.github.io/inventory-management/

Key Features
Manage inventory items efficiently
Add, update, and remove product information
Track available stock details
Search and organize inventory records
Responsive interface for different devices
Simple and fast user experience

DevOps Implementation

Docker

- Created a Dockerfile to containerize the application
- Built Docker images for consistent application execution
- Improved portability across different environments

Jenkins CI/CD Pipeline

- Created a Jenkinsfile to define and automate the CI/CD pipeline workflow.
- Configured Jenkins to pull source code from GitHub and execute build stages automatically.
- Improved development workflow by enabling continuous integration and faster deployment process.


Pipeline workflow:

1. Pull source code from GitHub repository
2. Build application environment
3. Execute automated pipeline stages
4. Validate project files
5. Prepare application for deployment
